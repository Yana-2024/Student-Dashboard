*Student Dashboard*: 
A simple dashboard built using HTML,CSS and JavaScript. It includes navigation bar, skill cards, course progress tracking, and a functional dark/light mode toggle using JavaScript.


# Features
Sidebar Navigation
Skills section 
Course Section 
Dark and Light mode toggle
Smooth hover animation
Responsive layout structure 

# Technologies Used
HTML
CSS (Flexbox, custom CSS variables, light/dark theme styling)
JavaScript
DOM Manipulation
LocalStorage to save mode preference
Font Awesome Icons
Google Material Icons
Google Fonts (Poppins)

# Purpose
The goal of the project was to practice layout design, styling, and introducing interactivity using JavaScript.
